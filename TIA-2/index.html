Trabajo PA/PIA

El propósito de este trabajo es diseñar e implementar una página web responsiva que no solo cumpla con los principios básicos de accesibilidad y usabilidad, sino que también brinde una experiencia interactiva y atractiva para los usuarios. Para lograr esto, se utilizarán tecnologías esenciales como HTML para la estructura del contenido, CSS para el diseño y la presentación visual, y JavaScript para agregar dinamismo e interactividad. Además, se integrará el framework Bootstrap, que facilitará la creación de un diseño adaptable a distintos dispositivos y tamaños de pantalla, garantizando una correcta visualización tanto en computadoras de escritorio como en dispositivos móviles.
